# ai-cards-demo 测试手册

10 桌 Demo 的测试场景与验证方法。

---

## 使用说明

- 访问 `http://<your-host>:<port>/` 进入 Demo 中心，点击对应行业卡片进入各桌
- 每桌有 3–4 个预设场景卡片，点击后自动填入输入框，或直接手动输入
- 点击"提交给 AI 助手"触发响应
- 通用标准：响应时间 < 10 秒，输出中文，有明确分段结构

## 桌1 · 油气能源 · 管理层

路径：`/oil-gas-mgmt`  
交互：点击左侧成本项目行触发 AI 分析；展开"管理周报生成"可手动输入。

### 场景1：成本结构分析

**点击**：报告编写人力（或直接输入下方内容）  
**输入**：帮我分析当前报告编写的人力成本结构，有哪些地方可以用 AI 降本？给出量化的改进空间。

| 验证点 | 预期结果 |
|---|---|
| 引用现状数据 | 65份/月、18人天/份、人工复核43%等数字出现 |
| 识别可AI化环节 | 至少指出数据整理/初稿生成/质量复核三个方向 |
| 给出量化降本空间 | 有具体的人天节省或成本节省估算 |
| 建议可落地 | 有具体实施路径或优先级 |

### 场景2：人员效率测算

**输入**：试点组 AI 辅助后产出提升了 87%，请帮我测算如果全院 42 名高级解释人员都引入 AI 辅助，每年可节省多少人力成本？

| 验证点 | 预期结果 |
|---|---|
| 引用试点数据 | 87% 提升、3.1→5.8 份/月数字出现 |
| 全院推广测算 | 基于 42 人给出年度人力节省数字 |
| 换算成本 | 有人力成本的货币化估算（含合理假设说明） |
| 风险提示 | 提到推广周期、培训成本等注意事项 |

### 场景3：计算资源优化

**点击**：智算集群运维（或直接输入下方内容）  
**输入**：智算中心集群平均利用率 61%，峰值 97%，帮我分析当前资源配置是否合理，有没有优化空间？

| 验证点 | 预期结果 |
|---|---|
| 判断配置合理性 | 指出 61% 均值 + 97% 峰值说明配置偏紧但不浪费 |
| 给出优化方向 | 弹性调度/错峰排队/资源预留策略 |
| 量化优化空间 | 有具体的利用率提升或成本节省估算 |
| 建议分短中期 | 有短期/中期行动项 |

### 场景4：管理周报起草

**展开**"管理周报生成"  
**输入**：帮我起草本周数智化项目进展管理周报，涵盖三个攻关组的进度、成本和下周计划。

| 验证点 | 预期结果 |
|---|---|
| 格式完整 | 有标题、日期、三个攻关组分节 |
| 覆盖三组 | AI+办公、AI+解释、智算中心数智化均有描述 |
| 结构规范 | 每组含本周进展、问题/风险、下周计划三项 |
| 数字引用 | 引用预算、人员等场景数据 |

## 桌2 · 油气能源 · 技术层

路径：`/oil-gas-tech`  
交互：点击左侧命令按钮触发终端输出；底部输入框可自定义查询。

### 场景1：井况异常诊断

**点击**：断层解释仲裁  
**或输入**：GS-B23 井电流波动 ±15A，泵效已降至 67%，帮我分析故障原因，给出是否需要立即作业的判断。

| 验证点 | 预期结果 |
|---|---|
| 引用具体数据 | ±15A 波动、67% 泵效数字出现 |
| 故障原因分析 | 给出 2–3 个可能原因（泵筒磨损/气锁/出砂等） |
| 作业建议明确 | 给出"建议作业"或"继续观察"的明确结论 |
| 高风险标注 | 涉及停井/作业决策有 ⚠️ 提示 |

### 场景2：规范查询

**点击**：规范查询  
**或输入**：帮我查询压裂施工规程要点，特别是排量和砂比的控制标准。

| 验证点 | 预期结果 |
|---|---|
| 排量标准 | 3.0–4.5m³/min 出现 |
| 砂比标准 | 不超过 35% 出现 |
| 停泵压力说明 | >15MPa 需复查出现 |
| 补充安全要点 | 有至少 2 条补充注意事项 |

### 场景3：产量递减分析

**点击**：智算作业调度  
**或输入**：GS-A17 井投产 3 年，初产 85 桶，当前 62 桶，递减率 9.8%/年，预测 12 个月后产量并给出稳产建议。

| 验证点 | 预期结果 |
|---|---|
| 预测数字合理 | 12 个月后产量约 56 桶（62×(1−9.8%)≈56） |
| 递减类型分析 | 说明递减类型（指数/双曲） |
| 稳产措施 | 给出注水/压裂/工艺优化等至少 2 条建议 |
| 经济效益提示 | 提到产量临界值或措施 ROI |

### 场景4：作业日报生成

**点击**：QC 报告生成  
**或输入**：今日 GS-B23 井下泵作业，8:00 开始，16:30 完成，更换泵筒和抽油杆，测试泵效 82%，帮我生成规范的作业日报。

| 验证点 | 预期结果 |
|---|---|
| 格式规范 | 有井号、日期、作业类型、起止时间 |
| 作业内容完整 | 更换泵筒、抽油杆均记录 |
| 测试结果记录 | 泵效 82% 正确记录 |
| 对比修复前 | 对比修复前 67% 泵效，说明提升效果 |
